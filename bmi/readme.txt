One simple way to calculate ideal weight is to use the BMI index. Given the given information, 
write a simple program that calculates the BMI of a person from given input datas and 
shows in the output whether this person is underweight, overweight, or fit.

bmi= weight/ height in meter**2

note that height in input is in centimeter.

If the BMI is below 18.5, it is considered underweight.
If the BMI is between 18.5 and 24.9, it is ideal.
If it is above 25, it is considered overweight.
